void StartReceiverTask ( void *  argument)

Function implementing the receiverTask thread.

Parameters

Return values
None

Receiver task.

This task continuously checks for new data and triggers the printing of the data on the LED matrix if both the updateLedMatrix flag and newDataAvailable flag are set. Once the data is printed, it clears the updateLedMatrix flag and sets the newDataAvailable flag.
